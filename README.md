# fin-text-style

토스나 국내 증권 앱처럼 한국어 금융 UI에서 자주 쓰이는 가격/등락률 텍스트를
일관된 형식으로 만들어 주는 TypeScript 유틸리티입니다.

예:
- `1000` → `"1,000원"`
- `(88300, 86000)` -> `"전일 대비 +2,300원 (+2.67%) 상승, 적당한 상승 흐름입니다."`

---

## What this library solves

실제 금융 서비스에서는 다음과 같은 작업을 반복해서 하게 됩니다.

- 숫자에 천 단위 콤마 붙이기 (`1234567 → 1,234,567`)
- 원/만 원 단위 표시
- 현재가와 전일가로 등락률 계산
- 상승/하락/보합에 따라 다른 텍스트/색을 보여주기
- “소폭 상승 / 급락”처럼 변동의 강도를 표현하기
- 화면 읽기 도구용(접근성) 텍스트 만들기

`fin-text-style`은 이런 로직을 한 곳에 모아두고,
UI 컴포넌트에서 바로 가져다 쓸 수 있는 형태의 API를 제공합니다.
